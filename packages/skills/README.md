# @allem-ui/skills

Agent Skills for Allem UI — helps AI coding assistants (Claude Code, Cursor, Codex) understand and use Allem UI components correctly.

## Install

```bash
npx skills add kingofmit/allem-ui
```

This copies Allem UI best practices and component API reference into your project, making them available as context for AI agents.

## What's included

- **SKILL.md** — Overview, setup, conventions, and quick examples
- **rules/button.md** — Button component API and patterns
- **rules/input.md** — Input & Textarea components
- **rules/forms.md** — Select, Checkbox, Radio, Switch, Slider
- **rules/modal.md** — Modal & Drawer components
- **rules/card.md** — Card component variants
- **rules/overlay.md** — Tooltip, Popover, Dropdown, Context Menu
- **rules/feedback.md** — Badge, Spinner, Skeleton, Toast
- **rules/navigation.md** — Tabs, Breadcrumbs, Pagination
- **rules/layout.md** — Box, Flex, Grid, Container, Divider
- **rules/typography.md** — Heading, Text, Code, Link
- **rules/data-display.md** — Avatar, Table, Accordion
- **rules/date-picker.md** — Calendar, DatePicker, DateRangePicker, TimeField
- **rules/data-grid.md** — DataGrid with sorting, filtering, pagination
- **rules/chat.md** — Chat UI components
- **rules/kanban.md** — Kanban board with drag-and-drop
- **rules/pricing.md** — Pricing tables and comparison
- **rules/changelog.md** — Changelog timeline and version badges
- **rules/theme.md** — Theme setup, dark mode, design tokens
