# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Allem UI, please report it responsibly.

**Do not open a public issue.** Instead, email [ahmed@allzemhub.com](mailto:ahmed@allzemhub.com) with:

- A description of the vulnerability
- Steps to reproduce
- Affected package(s) and version(s)
- Any potential impact

You will receive an acknowledgment within 48 hours. We will work with you to understand the issue and coordinate a fix before any public disclosure.

## Supported Versions

Security updates are applied to the latest release of each package.

## Disclosure Policy

Once a fix is available, we will publish a patch release and credit the reporter (unless anonymity is requested).
